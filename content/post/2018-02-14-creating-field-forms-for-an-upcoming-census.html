If you are preparing for an upcoming census you may need to print maps and field forms. To create the maps you can use the function maply_tag() – you can read the help file of maply_tag() here, and you can preview the effect of changing the default behaviors of this function using this online app). To create the field forms you can use an app available via the database team. The app looks like this:

App to create field forms. Access this app via the database team.

For the option “Header columns on all sheets” it is wisest to select “No”. The value “No” forces a header to appear at the top of every page instead of on its own page. This way you will save one page per quadrat, and thus you will avoid wasting money and paper.

This is an example of what you should avoided – which results from setting the option “Header columns on all sheets” to “Yes” (please don’t do this):

Example of what to avoid. To avoid wasting one page per quadrat please set the optoin ‘Header columns on all sheets’ to ‘No’.
